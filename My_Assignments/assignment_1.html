HTML TAGS AND THEIR FUNCTIONS

1. <!DOCTYPE>- Defines the document type

2. <html>- Defines an HTML document

3. <head>- Contains metadata/information for the document

4. <title>- Defines a title for the document

5. <a>- Defines a hyperlink

6. <abbr>- Defines an abbreviation or an acronym

7. <address>- Defines contact information for the author/owner of a document

8. <area>- Defines an area inside an image map

9. <article>- Defines an article

10. <aside>- Defines content aside from the page content

11. <audio>- Defines embedded sound content

12. <b>- Defines bold text

13. <base>- Specifies the base URL/target for all relative URLs in a document

14. <bdi>- Isolates a apart of text that might be formatted in a different direction from other text outside it

15. <blockquote>- Defines a section that is quoted from another source

16. <cite>- Defines the title of a work

17. <body>- Defines the document's body

18. <br>- Defines a single line break

19. <button>- Defines a clickable button

20. <canvas>- Used to draw graphics, on the fly, via scripting (usually JavaScript)

21. <caption>- Defines a table caption

22. <code>- Defines a piece of computer code

23. <col>- Specifies column within a colgroup element

24. <colgroup>- Specifies a group of one or more columns in a table for formatting

25. <data>- Adds a machine-readable translation of a given content

26. <datalist>- Specifies a list of pre-defined options for input controls

27. <dd>- Defines a description/value of a term in a description list

28. <del>- Defines text that has been deleted from a document

29. <details>- Defines additional details that the user can view or hide

30. <dfn>- Specifies a term that is going to be defined within the content

40. <dialog>- Defines a dialog box or window

41. <div>- Defines a section in a document

42. <dl>- Defines a description list

43. <dt>- Defines a term/name in a description list

44. <em>- Defines emphasized text

45. <embed>- Defines a container for an external application

46. <fieldset>- Group related elements in a form

47. <figcaption>- Defines a caption for figure element

48. <figure>- Specifies self-contained content

49. <footer>- Defines a footer for a document or section

50. <form>- Defines an HTML form for user input

51. <h1> to <h6>- Defines HTML headings

52. <header>- Defines a header for a document or section

53. <hgroup>- Defines a header and related content

54. <hr>- Defines a thematic change in the content

55. <i>- Defines a part of text in an alternate voice or mood

56. <iframe>- Defines an inline frame

57. <img>- Defines an image

58. <input>- Defines an input control

59. <ins>- Defines a text that has been inserted into a document

60. <kbd>- Defines keyboard input

61. <label>- Defines a label for an input element

62. <legend>- Defines a caption for a fieldset element

63. <li>- Defines a list item

64. <link>- Defines the relationship between a document and an external resource (most used to link to style sheets)

65. <main>- Specifies the main content of a document

66. <map>- Defines an image map

67. <mark>- Defines marked/highlighted text

68. <menu>- Defines an unordered list

69. <meta>- Defines metadata about an HTML document

70. <meter- Defines a scalar measurement within a known range(a guage)

71. <nav>- Defines navigation links

72. <noscript>- Defines an alternate content for users that do not support client-side scripts

73. <object>- Defines a container of an external application

74. <ol>- Defines an ordered list

75. <optgroup>- Defines a group of related options in a drop-down list

76. <option>- Defines an option in a drop-down list

77. <output>- Defines the result of calculation

78. <p>- Defines a paragraph

79. <param>- Defines a parameter for an object

80. <picture>- Defines a container for multiple image resources

81. <pre>- Defines preformatted text

82. <progress>- Represents the progress of a task

83. <q>- Defines a short quotation

84. <rp>- Defines what to show in browsers that do not support ruby annotations

85. <rt>- Defines an explanation/pronunciation of characters (for East Asian typography)

86. <ruby>- Defines a ruby annotation (for East Asian typography)

87. <s>- Defines text that is no longer correct

88. <samp>- Defines sample output from a computer program

89. <script>- Defines a client-side script

90. <search>- Defines a search section

91. <section>- Defines a section in a document

92. <select>- Defines a drop-down list

93. <small>- Defines smaller text

94. <source>- Defines multiple media resources for media elements (video and audio)

95. <span>- Defines a section in a document

96. <strong>- Defines important text

97. <style>- Defines style information for a document

98. <sub>- Defines subscripted text

99. <summary>- Defines a visible heading for a details element

100. <sup>- Defines superscripted text

101. <svg>- Defines a container for SVG graphics

102. <table>- Defines a table

103. <tbody>- Groups the body content in a table

104. <td>- Defines a cell in a table

105. <template>- Defines a container for content that should be hidden when the page loads

106. <textarea>- Defines a multiline input control (text area)

107. <tfoot>- Groups the footer content in a table

108. <th>- Defines a header cell in a table

109. <thead>- Group the header content in a table

110. <time>- Defines a specific time (or datetime)

111. <tr>- Defines a row in a table

112. <track>- Defines text tracks for media elements (video and audio)

113. <u>- Defines some text that is unarticulated and styled differently from normal text

114. <ul>- Defines an unordered list

115. <var>- Defines a variable

116. <video>- Defines embedded video content

117. <wbr>- Defines a possible line-break

118. <!--...--> Defines a comment

119. <bdo>- Overrides the current text direction

120. <acronym>- Defines an acronym

121. <applet>- Defines an embedded applet

122. <basefont>- Specifies a default color, size, and font for all text in a document

123. <big>- Defines big text

124. <center>- Defines centered text

125. <dir>- Defines a directory list

126. <font>- Defines font, color, and size for text

127. <frame>- Defines a window (a frame) in a frameset

128. <frameset>- Defines a set of frames

129. <noframes>- Defines an alternate content for users that do not support client-side scripts

130. <strike>- Defines strikethrough text

131. <bgsound>- It is used to play the soundtrack in the background

132. <isindex>- It is used to query any document through a text field.

133. <keygen>- It is used to specify a key-pair generator field in a form.

134. <marquee>- it is used to create scrolling text or images on a webpage.

135. <menuitem>- Defines a command or menu that the user can utilize from the popup item.

136. <nobr>- It is used to create a single line text, that does not matter how long the statement is.

137. <noembed>- It is used to show that the browser is not supported by <embed> tag.

138. <q>- It is used for short quotations.

139. <spacer>- It is used to create some white space

140. <tt>- It was used for marking Keyboard input

141. <xmp>- It is used to create any content in letter format
